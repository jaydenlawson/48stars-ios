48Stars - 4.8+ Stars Finder (Beginner)

Map search bottom bar. Granular steps.

Prerequisites

  1. Xcode App Store install/open
  2. developer.apple.com $99 enroll
  3. Google Console: Maps SDK iOS + Places API keys (restrict iOS bundle com.sidekick48stars)

1. Clone

mkdir ~/Desktop/48StarsApp
cd ~/Desktop/48StarsApp
git clone https://github.com/jaydenlawson/48stars-ios.git
cd 48Stars

2. Xcode New Project (5min)

  1. Open Xcode
  2. File > New > Project
  3. iOS tab > App > Next
  4. Product Name: 48Stars
  5. Team: Sidekick Pty Ltd
  6. Organization Identifier: com.sidekick
  7. Bundle Identifier: com.sidekick48stars
  8. Interface: SwiftUI
  9. Language: Swift
  10. Uncheck tests/Core Data/CloudKit
  11. Next > Save in 48Stars folder (checkmark)

3. Add Podfile & Install

  1. Terminal in 48Stars: pod install
  2. "Complete!"

4. Replace Files

  1. Delete ContentView.swift FourStarsApp.swift (move trash)
  2. New File > Swift File → paste repo ContentView.swift / 48StarsApp.swift / MapUIView.swift / MapViewModel.swift
  3. Save all

5. Signing

  1. Left blue 48Stars proj
  2. Signing tab
  3. Check Automatically manage
  4. Team: Sidekick Pty Ltd
  5. Bundle Identifier: com.sidekick48stars

6. Keys

  1. ContentView.swift: YOUR_GOOGLE_MAPS_IOS_SDK_API_KEY
  2. MapViewModel.swift: YOUR_GOOGLE_PLACES_API_KEY

7. Run

  1. Product > Clean Build Folder
  2. Simulator iPhone
  3. Cmd+R

8. TestFlight

  1. Product > Archive
  2. Organizer → Distribute App → App Store Connect → Upload
  3. appstoreconnect.apple.com New App "48Stars" bundle com.sidekick48stars
  4. TestFlight → Internal → Add email

Zap 2026-02-09. Zero skips. ⚡️
