Four Point Eight Stars ⭐ - Complete Beginner Guide

Find 4.8+ star businesses on map. Bottom search bar. Every click/field listed.

Prerequisites (One Time)

  1. Xcode: Cmd+Space "App Store" → "Xcode" → Get → Install (wait, ~12GB)
  2. Apple Developer: developer.apple.com/account → Start Free Trial → Pay $99 → Verify ID/card
  3. Google Keys: console.cloud.google.com → New Project "48Stars" → APIs & Services → Library → Enable "Maps SDK for iOS" + "Places API" → Credentials → + Create Credentials → API Key → iOS apps restrict bundle com.sidekick-four-point-eight → Copy 2 keys

1. Clone Repo

mkdir ~/Desktop/48StarsApp
cd ~/Desktop/48StarsApp
git clone https://github.com/jaydenlawson/48stars-ios.git
cd 48stars-ios

2. Xcode New Project (Granular)

Close other Xcode. Use Terminal for cd.
  1. Xcode open (Spotlight Cmd+Space "Xcode")
  2. File → New → Project
  3. Top iOS tab → App → Next
  4. Product Name: Four Point Eight Stars
  5. Team: dropdown "Sidekick Pty Ltd"
  6. Organization Identifier: com.sidekick
  7. Bundle Identifier auto com.sidekick.four-point-eight-stars → OK or edit com.sidekick-four-point-eight
  8. Interface: SwiftUI
  9. Language: Swift
  10. Uncheck Unit Tests / UI Tests / Core Data / CloudKit
  11. Next → Create in 48stars-ios folder (checkmark "Create Git repository no")

3. Pods

cd "Four Point Eight Stars"
pod install

"Pod installation complete!"

4. Replace Code Files

  1. Xcode left nav: trash ContentView.swift, Four_Point_Eight_StarsApp.swift
  2. File → New → File → iOS → Swift File → Next → 48StarsApp.swift** → Create
  3. Paste repo 48StarsApp.swift code → Save Cmd+S
  4. Repeat for ContentView.swift, MapUIView.swift, MapViewModel.swift (repo copies)

5. Signing (Screenshot Match)

  1. Left nav blue **Four Point Eight Stars** proj
  2. Signing & Capabilities** tab
  3. Check Automatically manage signing
  4. Team: "Sidekick Pty Ltd"
  5. Bundle Identifier: com.sidekick-four-point-eight
  6. Green "Xcode managed profile" + Apple Development Jayden Lawson cert

6. API Keys

  1. ContentView.swift line ~50: "YOUR_GOOGLE_MAPS_IOS_SDK_API_KEY" → Maps key
  2. MapViewModel.swift line ~20: "YOUR_GOOGLE_PLACES_API_KEY" → Places key
  3. Save all

7. Run

  1. Product → Clean Build Folder Cmd+Shift+K
  2. Toolbar simulator iPhone 15 Pro
  3. Play button Cmd+R (1st 2min)
  4. Map Sydney, bottom search "coffee" Enter → 4.8⭐ markers!

8. TestFlight Upload

  1. Product → Archive (Organizer pops)
  2. Organizer left list → latest Archive → Distribute App
  3. App Store Connect → Upload
  4. appstoreconnect.apple.com → My Apps → + → iOS "Four Point Eight Stars" SKU "48stars1" bundle com.sidekick-four-point-eight → Create
  5. TestFlight tab → Build list → latest → + Internal Testing → + your email → Submit
  6. TestFlight iPhone app → Install test build!

Errors?

  • Pods: pod deintegrate && pod install
  • Signing red: Team/bundle match
  • Map blank: Key restrict bundle
  • Clean: Cmd+Shift+K
